# Code of Conduct

Awale aims to be a respectful, low-friction project for contributors and users.

## Expectations

- Be constructive and professional.
- Critique ideas, not people.
- Keep discussions focused on the project.
- Assume good faith unless there is clear evidence otherwise.

## Unacceptable behavior

- Harassment, discrimination, threats, or abuse
- Personal attacks, trolling, or deliberate disruption
- Publishing private information without consent

## Reporting

If you need help or want to report a concern, open a GitHub issue and mention `Code of Conduct` in the title.
If the issue involves a maintainer or you prefer direct escalation, contact Emilio Dávola at `emidavola@gmail.com`.

Reports will be handled promptly, confidentially, and with a bias toward resolving the issue first.

## Scope

This code of conduct applies to issues, pull requests, and any other project-related communication.
